Introduction

I always wanted a documentation browser for developer, and dashdocs is wonderful, but there isn't a full mobile solution(maybe a ios one, but not a full cross hybrid)

Usefullness

  • 150+ Documentation(from Dashdocs)
  • 281 User Contribute Docs

Competition

There is LovelyDocs for android, but is using a very old webkit engine for rendering html, which cause html to load very slow and it is slow overall, and it does not have user contributed ones
